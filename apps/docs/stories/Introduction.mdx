    ## What is Prism?

    Unlike production-ready design systems like Material-UI or Ant Design, Prism prioritizes **clarity and learning** over features. Every architectural decision is documented with inline comments explaining the "why" behind the implementation.

    ### Core Philosophy

    - **Education First** - Every decision is documented to teach design system concepts
    - **Simple Architecture** - Clear patterns over clever abstractions
    - **Progressive Complexity** - Start simple, add complexity only when teaching a concept
    - **Real-World Patterns** - Industry-standard practices from established design systems
  </div>
